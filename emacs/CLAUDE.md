# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

Bare-bones Emacs configuration focused on essential development tooling. Uses built-in `package.el` (no straight.el despite README mentioning it) with ELPA/MELPA/NonGNU archives. All configuration lives in three files:

- `early-init.el` — GC tuning, UI chrome removal, frame defaults (runs before package init)
- `init.el` — all package configuration via `use-package` with lexical binding
- `custom.el` — auto-generated by Emacs Custom; do not edit by hand

## Key Architectural Decisions

**Evil + SPC leader**: Vim-style editing via `evil` with `SPC` as leader in normal/visual states. Leader prefix maps are defined as named prefix commands (`my/leader-buffer-map`, `my/leader-git-map`, etc.) and bound via `evil-define-key`.

**Completion stack**: vertico (UI) + orderless (matching) + marginalia (annotations) + consult (commands) + embark (actions). Corfu handles in-buffer completion (TAB-triggered, not auto).

**LSP via eglot**: Auto-starts for C, C++, Go, YAML. Events buffer disabled, progress reporting off.

**Tree-sitter**: All major modes remapped to `-ts-mode` variants via `major-mode-remap-alist`. Grammar sources defined in `treesit-language-source-alist` — install grammars with `M-x treesit-install-language-grammar`.

**Nerd Icons**: Used across corfu, dired, completion, and magit (`magit-format-file-nerd-icons`). Requires font installation: `nerd-icons-install-fonts`.

## Adding Packages

Follow the existing `use-package` pattern in `init.el`. Use `:ensure nil` for built-ins. Prefer `:defer t` and hook-based activation. Place new packages near related ones (e.g., tree-sitter modes near the `treesit-language-source-alist` block).

## Notes

- `magit` is declared `:ensure nil` — it must be installed separately or already present; the `custom.el` package list includes it so `package-install` handles it.
- `wgrep` has `wgrep-auto-save-buffer t` — edits in grep results are saved automatically.
- `corfu-auto` is `nil` — completion popup requires explicit TAB, not automatic.
- Theme is `modus-vivendi` (dark); `modus-operandi-tinted` is commented out as an alternative.
